# Oracle-Flower-Shop-Database
Design and implementation of a Flower Shop Management Database using Oracle SQL

## Overview
This project focuses on designing and implementing a relational database system
for managing a flower shop using Oracle Database.

## Objectives
- Manage customers, orders, products, inventory, and employees
- Ensure data integrity with constraints and triggers
- Support reporting and business analysis

## Technologies
- Database: Oracle Database
- Language: Oracle SQL / PL-SQL
- Tools: Oracle SQL Developer

## Database Design
- ERD and relational schema
- Normalized to 3NF

## Key Features
- Fully normalized relational database
- Constraints (PK, FK, CHECK, UNIQUE)
- Views for reporting
- Triggers for business rules
- Stored procedures and functions

## Main Modules
- Customer Management
- Order Management
- Inventory Management
- Employee Management
